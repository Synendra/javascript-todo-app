# About Todo App

built a simple todo list application using HTML,CSS and Javascript.This project allows users to add tasks, mark them as completed, and delete them.  
Tasks are saved in the browser using **LocalStorage** so they remain available even after refreshing the page.

---

## Features

- user can add new tasks
- mark tasks as completed
- can delete tasks
- implemented data saving using browswer localStorage bwtween reloads
